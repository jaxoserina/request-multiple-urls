To set the project up and running, follow steps below
1. Download the project folder and ensure node version is 10 or above
2. Go to the project folder using command prompt and install all dependency packages using the "npm i" command
3. Run "npm start" command from the console to start the server, express servers runs on port 3000.
4. Open a browser window and run http://localhost:3000/ to see the API json responses.
5. To run unit tests, run "npm test" command from the terminal.
